Documentatie

Inleiding

Voor het vak webapplicaties hebben wij de opdracht gekregen om een website te maken. De bedoeling is om beide de front-end en back-end volledig zelf uit te programmeren. Ik heb ervoor gekozen om een auto onderdelen webshop te maken. Zelf sleutel ik graag aan mijn eigen auto als hobby, dus het leek met wel leuk om een webshop te maken die daar ook mee te maken heeft. De website is gemaakt met HTML, CSS en JavaScript. De back-end maakt gebruik van een SQLite database, nodeJS en expressJS.

Class diagram

Hier boven is de class diagram te zien van de database. Het is een vrij simpele database met niet te veel tabellen en connecties. Op het moment doen de order_product en orders tabellen nog niks. Het toevoegen van daadwerkelijke orders had geen prioriteir voor mij, maar ik had voor de zekerheid al wel de tabellen dus aangemaakt.
